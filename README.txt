COMO USAR

1. Extraia o ZIP.
2. Deixe os arquivos na mesma pasta:
   - tag-editor-categorizado-manual.html
   - tags.json

3. Abra o terminal nessa pasta:
   python -m http.server 8000

4. Acesse:
   http://localhost:8000/tag-editor-categorizado-manual.html

Novidade:
- você pode selecionar tags categorizadas;
- você também pode digitar tags manuais;
- a tag manual entra no look selecionado;
- duplicados são bloqueados automaticamente.
